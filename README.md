## 🛠 Tecnologías

- HTML
- CSS
- JavaScript
- Node.js
- Express
- MongoDB Atlas
- Mongoose
- JWT
- Render
- GitHub Pages

---

## ✨ Funcionalidades

- Registro de usuarios
- Login
- Autenticación con JWT
- Crear tareas
- Marcar tareas completadas
- Eliminar tareas
- Persistencia en MongoDB

---

## ⚙️ Instalación local

```bash
git clone https://github.com/Agustin-Burkhard/todo-fullstack.git
cd todo-fullstack
npm install
```

Crear archivo `.env`

```env
MONGO_URI=tu_uri
JWT_SECRET=tu_secret
```

Iniciar servidor:

```bash
node index.js
```
